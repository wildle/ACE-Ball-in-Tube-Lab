\chapter{Laboratory Setup and Parameter Identification}

\begin{itemize}
    \item Description of the experimental setup: Plexiglas tube, ToF sensor, fan motor, Beckhoff PLC.
    \item Hardware constraints: maximum tube height, voltage limits, sensor range.
    \item ToF sensor calibration: offset correction and scaling procedure.
    \item Measurement process for fan step response.
    \item Parameter extraction: identification of \(k_\omega\), \(\tau_\omega\), and voltage offset \(u_0\).
    \item Computation of simplified model parameters \(\alpha_1\), \(\alpha_2\), \(\alpha_3\) from physical constants.
    \item Discussion of measurement accuracy and potential error sources.
\end{itemize}

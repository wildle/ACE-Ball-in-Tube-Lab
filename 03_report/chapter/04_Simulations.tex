\chapter{Simulation}

\begin{itemize}
    \item Implementation of the nonlinear ball and fan model in Simulink.
    \item Inclusion of actuator saturation and measurement noise.
    \item Computation of steady-state operating point for linearization.
    \item Linearization of the nonlinear model and derivation of the linear state-space system.
    \item Transfer function computation for PID controller tuning.
    \item PID controller design using the linear model and the pidTuner tool.
    \item Anti-windup configuration and handling of tube height limits.
    \item Optional output filtering to reduce measurement noise.
    \item Polynomial trajectory generation for smooth height transitions.
    \item Flatness-based feed-forward input computation and integration into the control loop.
    \item Structure of the full closed-loop Simulink simulation including plant, PID, feed-forward, trajectory generator, saturation blocks, and noise.
\end{itemize}
